## [ `Description` ]
> - A CLI application that converts binary numbers to decimal numbers and vice versa. <br />
    The technologies are Typescript and NodeJS.

<br />
<br />
<br />



## [ `Development` ]
> - Run the following commands.
```bash
yarn install
yarn build
yarn start
```
